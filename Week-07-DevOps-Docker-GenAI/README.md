# Week 07 — DevOps, Docker and Generative AI

## Week Overview

This week introduces modern engineering practices that connect development, delivery, and intelligent applications. The labs focus on DevOps automation, Docker containerization, and practical GenAI integration using ASP.NET Core 8.

---

## DevOps Concepts

- What is DevOps
- SDLC and delivery lifecycle
- CI/CD pipeline basics
- Git workflow and branching
- Build automation
- Release and deployment concepts
- Azure DevOps overview
- GitHub Actions

---

## Docker Concepts

- Docker architecture
- Images and containers
- Volumes and networks
- Dockerfile fundamentals
- Docker Compose orchestration
- Port mapping and environment variables

---

## GenAI Concepts

- Artificial intelligence foundations
- Machine learning and deep learning
- Generative AI and large language models
- Transformers and prompt engineering
- Embeddings, tokens, and context window
- Temperature, hallucinations, and RAG
- AI agents and responsible AI

## Learning Outcomes

By the end of Week 7, you will be able to:

- Explain the core principles of DevOps and CI/CD
- Build and publish .NET 8 applications with GitHub Actions
- Containerize ASP.NET Core applications using Docker
- Run multi-container systems with Docker Compose
- Describe the fundamentals of Generative AI and LLMs
- Build a simple AI-enabled web API using configuration-based integration
