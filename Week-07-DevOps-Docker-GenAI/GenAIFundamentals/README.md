# Generative AI Fundamentals

## Objective

Introduce the foundations of Artificial Intelligence, Machine Learning, Deep Learning, and Generative AI, then demonstrate a simple AI integration using ASP.NET Core 8.

## Concepts Covered

- Artificial Intelligence
- Machine Learning
- Deep Learning
- Generative AI
- Large Language Models
- Transformers
- Prompt Engineering
- Embeddings
- Tokens
- Context Window
- Temperature
- Hallucinations
- RAG
- AI Agents
- Responsible AI

## Prerequisites

- .NET 8 SDK
- Basic knowledge of ASP.NET Core Web APIs
- A free Gemini or OpenAI-compatible API key

## Installation

Add the API key to appsettings.json or set the corresponding environment variable before running the project.

## How to Run

Run the sample API and send a POST request to /api/ai/chat using Swagger or Postman.

## Expected Output

The API should return a generated response from the configured AI provider.

## Learning Outcomes

- Explain foundational AI concepts
- Understand LLM behavior and prompt design
- Build a simple AI-powered API endpoint
