# GenAI Lab 02 — Prompt Engineering

## Objective

Learn how carefully designed prompts can improve the quality and relevance of AI responses.

## Concepts Covered

- Prompt engineering
- Context window
- Tokens
- Temperature
- Hallucinations
- Responsible AI

## Folder Structure

```
GenAILab02-PromptEngineering/

```

## Prerequisites

- Familiarity with basic AI terminology

## Installation

No installation required.

## How to Run

Compare a simple prompt with a structured prompt and note the difference in output quality.

## Expected Output

You should observe that better prompts often produce more accurate and focused responses.

## Learning Outcomes

- Write clear and structured prompts
- Understand the effect of temperature and context window
- Evaluate AI outputs critically
