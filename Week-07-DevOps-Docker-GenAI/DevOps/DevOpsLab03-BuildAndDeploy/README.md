# DevOps Lab 03 — Build and Deploy

## Objective

Understand how a build moves through packaging, validation, and deployment stages in a production environment.

## Concepts Covered

- Build automation
- Release pipeline
- Deployment strategy
- Environment promotion

## Folder Structure

```
DevOpsLab03-BuildAndDeploy/
└── README.md
```

## Prerequisites

- Familiarity with CI/CD workflows
- Access to a deployment target or environment

## Installation

No installation required beyond your existing development environment.

## How to Run

Review the concepts in this lab and map them to the GitHub Actions workflow and the Docker deployment examples.

## Expected Output

You should be able to describe how build outputs are prepared and deployed to target environments.

## Learning Outcomes

- Understand release pipelines
- Explain deployment stages
- Relate build artifacts to deployment outcomes
