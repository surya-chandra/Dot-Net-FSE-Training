# DevOps Lab 01 — CI/CD Basics

## Objective

Learn the fundamentals of continuous integration and continuous delivery in a modern software workflow.

## Concepts Covered

- SDLC
- CI/CD pipeline
- Build automation
- Test automation
- Release and deployment

## Folder Structure

```
DevOpsLab01-CI-CD-Basics/
└── README.md
```

## Prerequisites

- Basic understanding of Git
- .NET 8 SDK

## Installation

No special installation is required beyond the .NET SDK and a code editor.

## How to Run

Review the workflow design concepts in this lab and compare them with the sample GitHub Actions workflow in the repository.

## Expected Output

You should be able to describe each stage of a CI/CD pipeline and relate it to a real build and deployment process.

## Learning Outcomes

- Describe the purpose of CI/CD
- Explain how automation improves delivery
- Understand the stages of a standard software release pipeline
