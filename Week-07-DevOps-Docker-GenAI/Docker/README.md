# Docker

## Objective

This section demonstrates how to package, run, and orchestrate .NET applications in containers using Docker.

## Concepts Covered

- Docker architecture
- Images and containers
- Volumes and networks
- Dockerfile
- Docker Compose
- Environment variables
- Port mapping

## Folder Structure

```
Docker/
├── DockerLab01-ContainerBasics/
├── DockerLab02-DockerizingDotNetAPI/
└── DockerLab03-DockerCompose/
```

## Prerequisites

- Docker Desktop
- .NET 8 SDK
- Basic familiarity with ASP.NET Core

## Installation

Install Docker Desktop and verify the service is running.

## How to Run

Build the sample image and start the container using the commands provided in the lab folders.

## Expected Output

You should see the API running in a container and accessible through mapped ports.

## Learning Outcomes

- Understand Docker images and containers
- Create a Dockerfile for an ASP.NET Core API
- Use Docker Compose for multi-container scenarios
