# Docker Lab 01 — Container Basics

## Objective

Understand the relationship between Docker images, containers, volumes, and networks.

## Concepts Covered

- Docker architecture
- Images
- Containers
- Volumes
- Networks

## Folder Structure

```
DockerLab01-ContainerBasics/
└── README.md
```

## Prerequisites

- Docker Desktop

## Installation

Install Docker Desktop and verify that the Docker engine is running.

## How to Run

Use the following commands to explore container basics:

```bash
docker images
docker ps
docker run hello-world
```

## Expected Output

You should see the hello-world container run successfully and report that Docker is functioning correctly.

## Learning Outcomes

- Explain the purpose of Docker images and containers
- Understand how Docker manages isolated runtime environments
- Use basic container commands
